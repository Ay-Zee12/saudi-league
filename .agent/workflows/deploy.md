description: How to deploy the Saudi League website to Vercel
---
// turbo-all
1. Open your terminal in the project folder.
2. Run `npx vercel@latest` to start the deployment process.
3. Follow the interactive prompts (Log in if required).
4. When finished, it will provide you with a `.vercel.app` link.
5. If you own `saudiproresults.com`, you can then add it in the Vercel dashboard under Settings > Domains.
